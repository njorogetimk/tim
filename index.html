PICHOBBY

Being passionate on python programming, web design and photography, pichobby was designed by Timothy Njoroge to combine all three on one platform. It's an ongoing project hosted at the github page.

Timothy is an Electirical and Telecommunications Engineering graduate from Multimedia University of Kenya. While in campus, he gained an interest in scientific programming using python libraries such as NumPy, SciPy and Matplotlib. Coupled with a few skills in web design and photography, he decided to merge them all and initiated this project.
